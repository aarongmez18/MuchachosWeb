2. Finalidad del Tratamiento

Los datos personales recabados a través del sitio web se utilizan exclusivamente para:

  • Atender consultas o solicitudes de información.
  • Gestionar actividades, eventos o comunicaciones relacionadas con la Agrupación.
  • Mantener la relación con socios, colaboradores o interesados.

No se utilizarán los datos para finalidades distintas ni se cederán a terceros sin consentimiento expreso.

3. Base Legitimadora

El tratamiento de tus datos se basa en tu consentimiento al enviar un formulario o contactar con nosotros.

4. Conservación de los Datos

Los datos se conservarán solo durante el tiempo necesario para cumplir con la finalidad por la que fueron recabados o mientras exista una relación activa con el interesado.

5. Comunicación de Datos a Terceros

No se comunicarán datos a terceros, salvo obligación legal o en los casos necesarios para prestar un servicio solicitado (por ejemplo, proveedores de alojamiento web bajo contrato de confidencialidad).

6. Derechos de las Personas Interesadas

Puedes ejercer tus derechos de:

  • Acceso
  • Rectificación
  • Supresión
  • Limitación del tratamiento
  • Oposición
  • Portabilidad

Enviando una solicitud escrita a [am.muchachosdeconsolacion@gmail.com] o a la dirección postal indicada.

7. Seguridad de los Datos

Implementamos las medidas técnicas y organizativas necesarias para garantizar la seguridad de los datos y evitar su alteración, pérdida o acceso no autorizado.

8. Cambios en la Política de Privacidad

La Agrupación podrá modificar la presente política para adaptarla a nuevas exigencias legislativas o mejoras del sitio web. Las modificaciones se publicarán siempre en esta misma sección.
